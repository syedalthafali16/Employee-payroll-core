# Employee Payroll Management System (Core Java Practice)

This project was created as a **practice project** to strengthen my understanding
of **Core Java concepts** and basic application structure.

The goal of this project is **learning**, not feature completeness.

---

## Concepts Practiced

- Core Java fundamentals
- Object-Oriented Programming (OOP)
- Inheritance and Polymorphism
- Exception handling
- File handling using Java I/O
- Separation of concerns (model, service, util)
- Basic Java Swing UI (for learning purpose)
